# Black Prince — Pomodoro Timer

Одностраничный Pomodoro-таймер без зависимостей. Работает на ПК и телефоне. Интерфейс на английском и русском (язык определяется по браузеру, переключается вручную).

A dependency-free single-page Pomodoro timer for desktop and mobile, in English and Russian.

**Живая версия / Live:** https://andreyzarembo.github.io/black_prince/

## Возможности

- Таймер по центру (минуты:секунды), старт/пауза кликом по таймеру, пробелом или кнопкой.
- Настройки сверху: тема (системная/тёмная/светлая), 11 цветовых пресетов («Помидор», «Терминал», «Неон»…), цвет фона, цвет акцента, 9 шрифтов (включая цифровой, пиксельный, рукописный, Comic Sans и сегментный LCD с эффектом погасших сегментов — [DSEG](https://github.com/keshikan/DSEG), лицензия OFL), масштаб.
- Пресеты длительности: 5, 10, 15, 20, 25, 30, 40, 45, 60, 90, 120 минут + свои значения через «+» (свои можно удалять крестиком).
- Кнопки «xx:00», «xx:15», «xx:30», «xx:45» — таймер до ближайшей четверти часа (текущего или следующего часа) с автозапуском.
- Все настройки сохраняются в URL как base64 (`?config=...`) — ссылку можно закинуть в закладки или поделиться ею.
- Звуковой сигнал и мигание по окончании, оставшееся время в заголовке вкладки; если вкладка не в фокусе — браузерное уведомление (разрешение запрашивается при первом старте).
- Локализация EN/RU: автоопределение по языку браузера, переключатель в панели, выбор сохраняется в ссылке.
- Встроенная справка по кнопке «?» — краткая документация на текущем языке.
- PWA: устанавливается на телефон/планшет/десктоп («На экран „Домой“» / «Установить приложение») и работает офлайн — сервис-воркер кеширует страницу, LCD-шрифт и иконки.

## Запуск локально

Просто откройте `index.html` в браузере, сервер не нужен.
